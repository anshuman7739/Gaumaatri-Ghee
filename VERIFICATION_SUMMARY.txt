📞 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Q: Server won't start?
A: Check if port 4000 is in use:
   lsof -i :4000
   kill -9 <PID>
   PORT=4000 npm start

Q: Admin dashboard won't load?
A: Make sure server is running:
   curl http://localhost:4000/api/admin/coupons
   If fails, restart server

Q: Orders not showing in viewer?
A: Create a test order first:
   curl -X POST http://localhost:4000/api/apply-coupon ...
   Then refresh the viewer page

Q: Tests failing?
A: Make sure server is running on port 4000
   Run with verbose: bash -x test-coupon-system.sh | head -50

Q: Where's my API code?
A: server.js, lines 800-1150
